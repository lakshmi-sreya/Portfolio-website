# Personal Portfolio Website

This repository contains my personal portfolio website, developed to present my profile, technical skills, and projects in a structured and professional manner.

## Project Overview
The portfolio was built using core frontend web technologies as part of my continuous learning and practical application of software development fundamentals. It includes dedicated sections for professional summary, projects, resume, and contact information.

The objective of this project is to create a clean, responsive, and accessible web presence suitable for professional and recruitment purposes.

## Key Features
- Responsive and intuitive user interface  
- Professional profile and summary section  
- Projects showcase with visual representation  
- Resume and contact information  
- Light and dark theme toggle  

## Technology Stack
- HTML  
- CSS  
- JavaScript

## Purpose and Learning Outcomes
- Apply frontend development fundamentals in a real-world project  
- Improve UI structuring and styling practices  
- Develop a professional online portfolio for career opportunities
